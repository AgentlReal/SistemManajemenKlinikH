API Documentation

Base URL: http://localhost:8000/api

Authentication
- POST /register
  - Parameters: name, username, password, role (receptionist, doctor, lab, cashier, manager)
  - Example: curl -X POST http://localhost:8000/api/register -H "Content-Type: application/json" -d '{"name":"John Doe","username":"johndoe","password":"password123","role":"manager"}'
- POST /login
  - Parameters: username, password
  - Example: curl -X POST http://localhost:8000/api/login -H "Content-Type: application/json" -d '{"username":"johndoe","password":"password123"}'
- POST /logout
  - Requires Authentication
  - Example: curl -X POST http://localhost:8000/api/logout -H "Authorization: Bearer <token>"
- GET /user
  - Requires Authentication
  - Example: curl http://localhost:8000/api/user -H "Authorization: Bearer <token>"
- GET /users
  - Requires Authentication
  - Example: curl http://localhost:8000/api/users -H "Authorization: Bearer <token>"
